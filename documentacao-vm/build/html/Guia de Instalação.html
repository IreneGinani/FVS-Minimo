Guia de Instalação¶

Antes de utilizar uma aplicação desejada, é necessário efetuar a seguinte instalação de programas:

  • Para a instalação do docker siga o tutorial disponível em: Instalação do Docker
  • Para instalação do docker-compose siga o tutorial disponível em: Instalação do docker-compose
  • Para instalação do docker-machine siga o tutorial disponível em: Instalação do docker-machine

Após a instalação é necessário fazer clone ou download no repositório que contém as aplicações:

  • Faça clone do seguinte repositório: Example-Application-Security
